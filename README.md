# 项目介绍
本项目仓库在Jttor和Pytorch框架下实现了DDIM论文复现，并在Cifar10数据集（使用Jittor和Pytorch封装好的导入）上对二者进行实验，本项目仓库主要包括以下内容：

 

 - [x] 环境搭建（Jittor and Pytorch）
 - [x] 模型架构（U-Net优化）
 - [x] 实验（对比Jittor和Pytorch两种框架下DDIM的效果）
 - [x] 总结（对Jittor的 ~~踩坑~~ 使用观感以及优势总结）
 - [x] 附录（包含训练脚本、采样脚本、测试脚本）
 - [x] 参考代码仓库（膜拜大神）
 - [x] 随笔（记录一点胡思乱想）
